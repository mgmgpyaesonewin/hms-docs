# ops

Operational artifacts — RCAs, runbooks, deploy checklists.

```
ops/
├── incidents/   postmortems / root-cause analyses
└── deploy/      UAT steps, deployment checklists
```

For live services, see the `ops/` folder inside the service's design
tree (e.g. `summary-service/ops/runbook.md`). This top-level `ops/`
tree is for cross-cutting / repo-wide operational docs only.
